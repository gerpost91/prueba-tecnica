Informacion de Esta Applicación


Esta es una aplicaicon de Ejemplo, que consume una API basica con metodos CRUD, permite registrar nuevos usuarios asi mismo, permite realizar un login, el cual esta verificado por un JWT este es alamacenado en el local Storage para posteriormente validar por medio GUARDS la navegación en esta aplicaicon, asi mismo el JWT es decodificado y guardado en local Storage para utilizar su informacion en diversos componentes incluyento cache en formularios.

Los formularios en esta aplicacion han sido validados y tambien cuenta con un checkbox de seguridad para que el usuario no envié los datos por error.

Como extra esta aplicacion cuenta con la caracteristica que soporta muchos temas para que el usuario los personalice a su gusto, estos cambios tambien son persistentes y guardados en local Storage por medio de un servicio. Para la realizacion de esta aplicacion he utilizado Angular 9, Bootstrap y como base un template de Wrap Pixel.
